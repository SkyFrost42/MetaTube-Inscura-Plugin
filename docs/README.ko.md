# MetaTube Inscura 플러그인

언어: [English](../README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | [日本語](README.ja.md) | 한국어

이 플러그인은 Inscura용 MetaTube 메타데이터 스크래퍼 플러그인입니다. MetaTube 호환 API를 통해 작품 및 배우 메타데이터를 가져옵니다.
플러그인 자체는 메타데이터 서비스를 제공하지 않으며 사이트 데이터를 내장하고 있지 않습니다. 사용자가 설정한 MetaTube API 서비스로 Inscura의 스크래핑 요청을 전달하는 역할만 합니다.

이 플러그인은 Inscura 플러그인 규격에 따라 구현되었으며 현재 다음을 지원합니다.

- 작품 검색 및 작품 상세 스크래핑
- 배우 검색 및 배우 상세 스크래핑

## 기능

이 플러그인은 `movie` 와 `actor` capability를 모두 선언하므로 Inscura에서 다음과 같은 용도로 사용할 수 있습니다.

- 품번, 제목 또는 키워드로 작품 검색
- 제목, 요약, 감독, 배우, 평점, 출시일, 커버 이미지, 샘플 이미지, 프리뷰 비디오를 포함한 작품 상세 정보 가져오기
- 배우 검색
- 별칭, 생일, 키, 신체 치수, 국적, 아바타, 사진을 포함한 배우 상세 정보 가져오기

## 설치 방법

1. 프로젝트의 [GitHub Releases](https://github.com/SkyFrost42/MetaTube-Inscura-Plugin/releases) 페이지에서 공식 배포된 플러그인 `.zip` 패키지를 다운로드합니다.
2. Inscura의 플러그인 관리 화면에서 해당 `.zip` 패키지를 설치합니다.
3. 설치 후 플러그인 설정의 `API Base URL` 을 입력합니다.
4. 테스트를 실행한 뒤 플러그인을 활성화합니다.

테스트가 성공하면 Inscura가 설정된 MetaTube 서비스에 접근할 수 있다는 뜻입니다.

## 설정

현재 이 플러그인에는 필수 설정 항목이 하나만 있습니다.

- `apiUrl`: MetaTube API의 기본 URL

예시:

```text
https://your-metatube-server.com
```

서비스 루트 URL을 입력하고 `/v1` 은 직접 붙이지 마세요. 플러그인이 다음 엔드포인트를 자동으로 호출합니다.

- `/v1/movies/search?q=...`
- `/v1/movies/{provider}/{id}`
- `/v1/actors/search?q=...`
- `/v1/actors/{provider}/{id}`

## 사용 방법

설치와 설정이 끝나면 Inscura에서 바로 이 플러그인을 사용할 수 있습니다.

1. 플러그인 관리에서 MetaTube 플러그인을 활성화합니다.
2. `API Base URL` 이 정상적으로 접근 가능한지 확인합니다.
3. 작품 또는 배우 스크래핑 화면에서 이 플러그인을 선택합니다.
4. 검색어를 입력합니다.
5. 올바른 결과를 선택하고 메타데이터를 기록합니다.

## 테스트 및 활성화

이 플러그인은 `test(ctx)` 를 구현합니다. Inscura가 플러그인을 테스트하거나 활성화할 때 먼저 설정된 `apiUrl` 의 루트에 요청을 보냅니다.

즉, 다음 조건이 필요합니다.

- `apiUrl` 은 비어 있으면 안 됩니다
- 대상 서비스는 Inscura가 실행되는 환경에서 접근 가능해야 합니다
- 루트 URL은 성공 상태 코드를 반환해야 하며, 그렇지 않으면 플러그인 테스트가 실패합니다

테스트 실패의 일반적인 원인은 URL 입력 오류, 서비스 미실행, 접근 불가, 잘못된 경로입니다.

## 주의사항

- 이 플러그인은 원격 MetaTube 서비스에 의존하므로 오프라인에서는 동작하지 않습니다.
- 이 플러그인은 웹사이트를 직접 스크래핑하지 않습니다. MetaTube 호환 API가 반환하는 구조화된 데이터에 의존합니다.
- Releases 페이지의 공식 배포 패키지만 설치하고, 출처가 불분명하거나 재배포된 빌드는 피하는 것을 권장합니다.
- `apiUrl` 은 `https://example.com/v1` 이 아니라 `https://example.com` 과 같은 사이트 루트여야 합니다.
- 작품 및 배우 데이터의 최종 품질은 연결한 MetaTube 서비스와 그 상위 데이터 소스에 따라 달라집니다.
- 서비스가 2xx 이외의 상태 코드를 반환하면 플러그인은 이를 실패로 처리합니다.
- 서버 응답에 필드가 누락되면 플러그인은 빈 문자열, 빈 배열 또는 `0` 으로 대체하므로 일부 항목은 정보가 불완전할 수 있습니다.
- 이 플러그인은 Inscura API Version 1을 대상으로 합니다.

## 규정 준수 및 법적 위험 고지

- 이 플러그인은 합법적인 미디어 정보 정리, 색인화 및 메타데이터 관리 목적으로만 사용되어야 합니다.
- 대상 MetaTube 서비스, 상위 데이터 소스 및 관련 미디어 콘텐츠에 대한 적법한 접근권과 이용권이 있는지 사용자가 직접 확인해야 합니다.
- 사용 지역의 법률 및 규정과 대상 서비스의 이용 약관, 저작권 규칙, 개인정보 규칙, 데이터 스크래핑 정책을 준수하세요.
- 저작권, 개인정보, 초상권 또는 기타 권리를 침해하는 콘텐츠를 획득, 정리, 배포 또는 재배포하는 데 이 플러그인을 사용하지 마세요.
- 관련 콘텐츠가 사용 지역에서 연령 제한, 콘텐츠 등급, 네트워크 접근 또는 배포 제한의 대상이라면, 준수 여부 판단과 사용 책임은 사용자에게 있습니다.
- 플러그인 작성자와 유지관리자는 제3자 서비스가 반환하는 데이터의 적법성, 완전성, 가용성 또는 지속성에 대해 보증하지 않습니다.
- 이 플러그인에는 보호된 미디어 콘텐츠가 포함되어 있지 않으며, 결제 장벽, 인증, 지역 제한 또는 기타 접근 제어를 우회하는 기능도 제공하지 않습니다.

이 고지는 법률 자문이 아닙니다. 저작권, 콘텐츠 준수 또는 국경 간 접근 위험이 포함된 사용 사례라면 전문 법률 자문을 먼저 받으세요.

## 반환 데이터 범위

작품 데이터에는 일반적으로 다음이 포함됩니다.

- 기본 식별자: `id`, `provider`, `number`
- 텍스트 정보: `title`, `summary`, `director`
- 관련 정보: `actors`, `maker`, `label`, `series`, `genres`
- 미디어 자산: `thumbUrl`, `coverUrl`, `bigThumbUrl`, `bigCoverUrl`, `previewVideoUrl`, `photos`
- 기타 정보: `score`, `releaseDate`

배우 데이터에는 일반적으로 다음이 포함됩니다.

- 기본 식별자: `id`, `provider`, `name`
- 개인 정보: `aliases`, `birthday`, `height`, `measurements`, `nationality`
- 미디어 자산: `avatar`, `photos`

## 대상 사용자

이 플러그인은 다음 사용자에게 적합합니다.

- 이미 MetaTube 호환 서비스를 운영 중이거나 접근 가능한 Inscura 사용자
- Inscura에서 작품과 배우 스크래핑에 MetaTube 데이터를 함께 사용하려는 사용자
- Inscura를 MetaTube 커뮤니티 생태계와 연결하려는 사용자

## 알려진 제한 사항

- 현재 추가적인 고급 필터 설정은 제공하지 않습니다.
- 이 플러그인은 계정 로그인, 봇 방지 우회, 프록시 설정 또는 서비스 배포를 처리하지 않습니다.
- 배우 상세 정보의 설명, 데뷔일, 기술, 소셜 링크는 현재 플러그인이 채우지 않습니다.
- 실제 사용 가능한 필드는 서버 응답에 따라 달라지며 모든 항목에 대해 완전함이 보장되지 않습니다.
