# MetaTube Inscura プラグイン

言語: [English](../README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | 日本語 | [한국어](README.ko.md)

これは Inscura 向けの MetaTube メタデータスクレイパープラグインです。MetaTube 互換 API を通じて作品および出演者のメタデータを取得します。
このプラグイン自体はメタデータサービスを提供せず、サイトデータも内蔵していません。Inscura のスクレイピング要求を、ユーザーが設定した MetaTube API サービスへ転送するだけです。

このプラグインは Inscura のプラグイン仕様に従って実装されており、現在は以下をサポートします。

- 作品検索と作品詳細の取得
- 出演者検索と出演者詳細の取得

## 機能

このプラグインは `movie` と `actor` の両方の capability を宣言しているため、Inscura では次の用途に利用できます。

- 品番、タイトル、キーワードによる作品検索
- タイトル、概要、監督、出演者、評価、発売日、カバー、サンプル画像、プレビュー動画を含む作品詳細の取得
- 出演者検索
- 別名、生年月日、身長、スリーサイズ、国籍、アバター、写真を含む出演者詳細の取得

## インストール方法

1. プロジェクトの [GitHub Releases](https://github.com/SkyFrost42/MetaTube-Inscura-Plugin/releases) ページから正式リリースされたプラグイン `.zip` パッケージをダウンロードします。
2. Inscura のプラグイン管理画面からその `.zip` パッケージをインストールします。
3. インストール後、プラグイン設定の `API Base URL` を入力します。
4. テストを実行し、プラグインを有効化します。

テストが成功すれば、Inscura が設定済みの MetaTube サービスへ到達できていることを意味します。

## 設定

現在、このプラグインに必要な設定項目は 1 つだけです。

- `apiUrl`: MetaTube API のベース URL

例:

```text
https://your-metatube-server.com
```

サービスのルート URL を入力し、`/v1` は手動で追加しないでください。プラグインは次のエンドポイントを自動で呼び出します。

- `/v1/movies/search?q=...`
- `/v1/movies/{provider}/{id}`
- `/v1/actors/search?q=...`
- `/v1/actors/{provider}/{id}`

## 使用方法

インストールと設定が完了すると、Inscura でこのプラグインを直接利用できます。

1. プラグイン管理で MetaTube プラグインを有効にします。
2. `API Base URL` に正常にアクセスできることを確認します。
3. 作品または出演者のスクレイピング画面でこのプラグインを選択します。
4. キーワードを入力して検索します。
5. 正しい結果を選び、メタデータを書き込みます。

## テストと有効化

このプラグインは `test(ctx)` を実装しています。Inscura がプラグインをテストまたは有効化するとき、まず設定された `apiUrl` のルートへアクセスします。

つまり、次の条件が必要です。

- `apiUrl` は空にできません
- 対象サービスは Inscura が動作している環境から到達可能である必要があります
- ルート URL は成功ステータスコードを返す必要があります。そうでない場合、プラグインのテストは失敗します

テスト失敗の一般的な原因は、URL の入力ミス、サービス停止、到達不能、または誤ったパスです。

## 注意事項

- このプラグインはリモートの MetaTube サービスに依存しており、オフラインでは動作しません。
- このプラグインは Web サイトを直接スクレイピングしません。MetaTube 互換 API が返す構造化データに依存します。
- インストールには Releases ページの正式リリースパッケージのみを使用し、不明な配布物や再配布ビルドは避けることを推奨します。
- `apiUrl` には `https://example.com/v1` ではなく、`https://example.com` のようなサイトのルート URL を指定してください。
- 作品や出演者の最終的なデータ品質は、接続先の MetaTube サービスとその上流データソースに依存します。
- サービスが非 2xx ステータスを返した場合、プラグインは失敗として扱います。
- サーバー応答にフィールド欠落がある場合、プラグインは空文字列、空配列、または `0` にフォールバックするため、一部の項目は不完全になることがあります。
- このプラグインは Inscura API Version 1 を対象としています。

## 返されるデータ

作品データには通常、以下が含まれます。

- 基本識別子: `id`, `provider`, `number`
- テキスト情報: `title`, `summary`, `director`
- 関連情報: `actors`, `maker`, `label`, `series`, `genres`
- メディア資産: `thumbUrl`, `coverUrl`, `bigThumbUrl`, `bigCoverUrl`, `previewVideoUrl`, `photos`
- その他: `score`, `releaseDate`

出演者データには通常、以下が含まれます。

- 基本識別子: `id`, `provider`, `name`
- 個人情報: `aliases`, `birthday`, `height`, `measurements`, `nationality`
- メディア資産: `avatar`, `photos`

## 対象ユーザー

このプラグインは次のユーザーに適しています。

- すでに MetaTube 互換サービスを運用している、または利用可能な Inscura ユーザー
- Inscura で作品と出演者の両方に MetaTube データを使いたいユーザー
- Inscura を MetaTube コミュニティのエコシステムと連携させたいユーザー

## 既知の制限

- 現時点では追加の高度なフィルタ設定はありません。
- このプラグインは、アカウントログイン、Bot 対策、プロキシ設定、サービスのデプロイを扱いません。
- 出演者詳細の説明、デビュー日、スキル、ソーシャルリンクは現在プラグインでは補完されません。
- 実際に利用可能なフィールドはサーバー応答に依存し、すべての項目で完全とは限りません。
